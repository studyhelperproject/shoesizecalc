# ドローイングアプリ

これは、ユーザーがキャンバスに図形を描画し、マウスやタッチ入力で移動、拡大縮小、回転させることができるFlutterアプリケーションです。また、キャンバスの背景として画像をアップロードしたり、画像上の点の座標を取得したりすることもできます。

## 機能

*   **お絵かきキャンバス**: ユーザーが図形を描画し、操作できるキャンバスです。
*   **図形操作**:
    *   **描画**: キャンバスの何もない領域をドラッグして、新しい四角形を描画します。
    *   **移動**: 図形の内側をクリックしてドラッグし、移動させます。
    *   **リサイズ/変形**: 図形の頂点をドラッグして、形を変更します。
*   **カスタム背景**: 画像をアップロードしてキャンバスの背景として設定します。

## 使い方

1.  **アプリケーションを実行します。**
2.  **図形を描画する**: キャンバスの何もない部分をクリックしてドラッグします。四角形が描画されます。
3.  **図形を移動する**: 既存の図形の内側をクリックし、新しい位置までドラッグします。
4.  **図形を変形する**: 図形の角（頂点）の1つをクリックし、新しい位置までドラッグします。これにより、台形のような任意の四角形を作成できます。
5.  **背景画像を設定する**: アプリバーの画像アイコンをクリックして、ギャラリーから背景として使用する画像を選択します。

## 開発セットアップ

### 前提条件

*   Flutter SDK: Flutter SDKがインストールされていることを確認してください。[公式Flutterウェブサイト](https://flutter.dev/docs/get-started/install)で手順を確認できます。
*   Visual Studio CodeやAndroid Studioのようなエディタと、Flutterプラグイン。

### はじめに

1.  **リポジトリをクローンします:**
    ```bash
    git clone https://github.com/your-username/drawing-app.git
    cd drawing-app
    ```

2.  **依存関係をインストールします:**
    ```bash
    flutter pub get
    ```

3.  **アプリを実行します:**
    ```bash
    flutter run
    ```

## テスト

アプリケーションの品質を保証するためには、堅牢なテスト戦略が不可欠です。このプロジェクトには、単体テスト、ウィジェットテスト、統合テストを含むテスト環境がセットアップされています。

### すべてのテストを実行する

プロジェクト内のすべての単体テストとウィジェットテストを実行するには、単に次のコマンドを実行します:

```bash
flutter test
```

統合テストを含むすべてのテストを実行するには、各テストタイプのコマンドを順番に実行します。
