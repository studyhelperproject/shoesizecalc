## Project Goal

*   Create a Flutter application that allows users to draw shapes on a canvas, move, scale, and rotate them. Users can also upload an image as a background and get the coordinates of points on the image.

## Key Features

*   Drawing Canvas
*   Shape Manipulation (Move, Scale, Rotate)
*   Custom Background Image
*   Coordinate System

## Overall Architecture

*   Flutter-based application.
*   The core logic is implemented in Dart.
*   The UI is built using Flutter widgets.
